# dotfiles

This repo contains dotfiles managed with stow. The command to make symlinks is the makesymlinks.sh

This repo has several branches. The main branch contains configurations that are the same between all my computers. There branches for each computer with configs that are different.

My neovim configuration has been moved to a separate repository. You can find it [here.](https://github.com/lcroberts/nvim-config)

I also have a repo I use for my nixos configuration. It can be found [here.](https://github.com/lcroberts/nixos)
